# Changelog

All notable changes to this project will be documented in this file.

## [2.0.1] - 2026-03-17

### Added
- CMake build system with preset support
- CMakePresets.json for multi-platform configuration
- Support for Windows, macOS, iOS, Android, HarmonyOS platforms
- C++17 standard requirement
- Compiler warning as error (-Werror /WX)
- Toolchain files for iOS and HarmonyOS cross-compilation
- CMake package config for installation
- Unit tests integration with GoogleTest
